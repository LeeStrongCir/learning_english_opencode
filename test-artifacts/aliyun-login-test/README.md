# 阿里云登录功能测试项目

## 项目概述
本项目针对阿里云（https://account.aliyun.com）登录功能进行全面的自动化测试。

## 测试目标
- 验证登录功能的正确性和稳定性
- 覆盖正常登录、异常场景、安全性测试
- 提供完整的测试报告和缺陷分析

## 测试范围
- 登录页面UI验证
- 正常登录流程（账号密码登录）
- 异常登录场景（错误密码、空字段、账号不存在等）
- 安全性测试（SQL注入、XSS、暴力破解防护）
- 验证码功能测试
- 多浏览器兼容性测试

## 测试环境
- 浏览器：Chrome（自动化测试）
- 测试框架：Playwright + pytest
- 测试数据：使用测试专用账号
