## TC001 - 正常搜索流程（完整端到端）
| 属性 | 值 |
|------|-----|
| 优先级 | P0 |
| 用例类型 | 功能测试 |
| 前置条件 | 1. 网络连接正常 2. 浏览器已打开（Chrome 最新版） 3. 屏幕分辨率 1920x1080 |
| 测试步骤 | 1. 在浏览器地址栏输入 https://www.baidu.com 并回车<br>2. 等待百度首页完全加载<br>3. 点击搜索框使其获得焦点<br>4. 输入搜索关键词"阿里巴巴"<br>5. 点击"百度一下"按钮<br>6. 等待搜索结果页面加载完成<br>7. 在搜索结果中查找带有"官方"标识的链接<br>8. 点击阿里巴巴官网链接<br>9. 等待目标页面加载完成<br>10. 验证页面 URL 和标题 |
| 测试数据 | 搜索关键词：阿里巴巴<br>预期官网URL：https://www.alibaba.com 或 https://www.alibaba.com.cn<br>官方标识文本："官方" |
| 预期结果 | 1. 百度首页在 3 秒内加载完成，搜索框可见可操作<br>2. 搜索框中正确显示"阿里巴巴"<br>3. 搜索结果页面在 5 秒内加载完成<br>4. 搜索结果中包含带有"官方"标识的阿里巴巴官网链接<br>5. 点击后成功跳转到阿里巴巴官网<br>6. 官网 URL 包含 alibaba.com<br>7. 官网页面标题包含"阿里巴巴"字样<br>8. 官网页面内容正常显示，无错误提示 |
| 关联需求 | F001, F002, F003, F004, F005, F006, F007, SC001, SC003, SC004, NF001, NF003 |
