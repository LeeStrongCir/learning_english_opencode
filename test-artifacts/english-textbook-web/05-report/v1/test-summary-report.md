## 失败用例详细分析
### 失败模式
所有 10 个失败用例均呈现相同的失败模式：

| 项目 | 详情 |
|------|------|
| 断言方法 | `to_have_class("active")` |
| 期望值 | `"active"` |
| 实际值 | `"nav-btn active"`, `"volume-btn active"`, `"unit-btn active"` 等 |
| 失败原因 | Playwright 的 `to_have_class` 默认进行精确匹配，当元素包含多个 CSS 类时断言失败 |
| 产品影响 | **无** - 产品功能正常，active 类已正确应用 |
