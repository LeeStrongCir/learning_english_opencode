## 🔍 失败分析

### 失败原因分类
| 原因类型 | 数量 | 说明 |
|----------|------|------|
| 测试断言问题 | 10 | `to_have_class("active")` 精确匹配失败 |
| 产品缺陷 | 0 | 无 |
| 环境问题 | 0 | 无 |

### 失败详情
所有10个失败用例均为同一问题：
- **问题**：Playwright `to_have_class("active")` 期望精确匹配
- **实际值**：`"nav-btn active"`, `"volume-btn active"`, `"unit-btn active"`
- **期望值**：`"active"`
- **结论**：**产品功能正常，active类已正确应用，仅测试断言需修复**

### 修复建议
```python
# 修改前
expect(element).to_have_class("active")

# 修改后
expect(element).to_have_class(re.compile(r".*\bactive\b.*"))
```
